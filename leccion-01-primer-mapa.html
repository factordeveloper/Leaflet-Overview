🗺️ Tu Primer Mapa con Leaflet

¡Bienvenido al curso de Leaflet! En esta primera lección aprenderás qué es Leaflet, cómo incluirlo en tu proyecto y cómo crear tu primer mapa interactivo. Leaflet es una biblioteca JavaScript de código abierto para mapas interactivos, ligera (solo ~42KB) y muy fácil de usar.

📍 Tu Primer Mapa

¡Este es tu primer mapa! Está centrado en Bogotá, Colombia. Puedes hacer zoom con la rueda del ratón y arrastrarlo para moverte.

💻 Código Explicado

Paso 1: Incluir Leaflet

Primero necesitas incluir el CSS y JavaScript de Leaflet en tu HTML:

<!-- CSS de Leaflet --> <link rel="stylesheet" href="leaflet.css" /> <!-- JavaScript de Leaflet --> <script src="leaflet.js"></script>

Paso 2: Crear el contenedor

Necesitas un div con un id y altura definida:

<div id="mapa" style="height: 400px;"></div>

Paso 3: Inicializar el mapa

Usamos JavaScript para crear el mapa:

// Crear el mapa centrado en Bogotá const mapa = L.map('mapa').setView( [4.5981, -74.0760], // Coordenadas [lat, lng] 13 // Nivel de zoom (1-18) ); // Añadir capa de mosaicos (tiles) L.tileLayer( 'https://tile.openstreetmap.org/{z}/{x}/{y}.png', { attribution: '© OpenStreetMap' } ).addTo(mapa);
💡 Tip: El nivel de zoom va de 1 (mundo entero) a 18 (máximo detalle). Las coordenadas usan el formato [latitud, longitud].

📚 Conceptos Clave

¿Qué es L?

  • L es el objeto global de Leaflet. Todas las funciones empiezan con L.
  • L.map() crea una instancia del mapa en el div especificado.
  • setView() establece el centro y zoom inicial del mapa.
  • L.tileLayer() añade las imágenes del mapa (llamadas "tiles" o mosaicos).
  • addTo() añade cualquier elemento al mapa.

¿Qué son los Tiles?

  • Los mapas se dividen en pequeñas imágenes cuadradas llamadas "tiles".
  • Se cargan dinámicamente según la zona que estés viendo.
  • OpenStreetMap es gratuito y de código abierto.
  • Existen otros proveedores: Mapbox, CartoDB, Stamen, etc.
