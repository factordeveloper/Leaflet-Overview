📍 Marcadores y Popups

En esta lección aprenderás a añadir marcadores (markers) al mapa para señalar ubicaciones específicas, y a mostrar información mediante popups y tooltips. Los marcadores son fundamentales para cualquier aplicación de mapas.

🗺️ Mapa con Marcadores

🎮 Prueba interactiva:

Haz clic en los marcadores para ver sus popups.

💻 Código: Marcador Básico

Crear un marcador simple

Un marcador señala un punto específico en el mapa:

// Crear un marcador en coordenadas específicas const marcador = L.marker([40.4168, -3.7038]); // Añadirlo al mapa marcador.addTo(mapa);

Añadir un Popup

Los popups muestran información al hacer clic:

// Marcador con popup L.marker([40.4168, -3.7038]) .addTo(mapa) .bindPopup('<b>Madrid</b><br>Capital de España') .openPopup(); // Abre el popup automáticamente

Añadir un Tooltip

Los tooltips aparecen al pasar el ratón:

// Marcador con tooltip L.marker([40.4530, -3.6883]) .addTo(mapa) .bindTooltip('Aeropuerto de Barajas', { permanent: false, // Solo al pasar ratón direction: 'top' // Posición del tooltip });
💡 Tip: Puedes usar HTML dentro de los popups para formatear el contenido con negritas, imágenes, enlaces, etc.

🔧 Opciones de Popup

L.marker([40.42, -3.70]) .addTo(mapa) .bindPopup('Contenido del popup', { maxWidth: 300, // Ancho máximo en píxeles minWidth: 100, // Ancho mínimo closeButton: true, // Mostrar botón cerrar autoClose: true, // Cerrar al abrir otro closeOnClick: true, // Cerrar al clic en mapa className: 'mi-popup' // Clase CSS personalizada });

Popup con HTML enriquecido

const contenidoHTML = ` <div style="text-align: center;"> <h3>🏛️ Museo del Prado</h3> <p>Uno de los museos más importantes</p> <a href="#" target="_blank">Más info</a> </div> `; L.marker([40.4138, -3.6921]) .addTo(mapa) .bindPopup(contenidoHTML);

📚 Opciones de Marcador

L.marker([40.42, -3.70], { draggable: true, // Arrastrable opacity: 0.8, // Transparencia (0-1) title: 'Mi marcador', // Tooltip nativo alt: 'Descripción', // Texto alternativo riseOnHover: true // Elevar al pasar ratón }).addTo(mapa);

Métodos útiles del marcador

  • marcador.getLatLng() - Obtener coordenadas
  • marcador.setLatLng([lat, lng]) - Mover marcador
  • marcador.remove() - Eliminar del mapa
  • marcador.openPopup() - Abrir popup
  • marcador.closePopup() - Cerrar popup
  • marcador.setOpacity(0.5) - Cambiar opacidad
💡 Tip: Los marcadores arrastrables (draggable: true) son útiles para permitir que el usuario seleccione una ubicación en el mapa.
