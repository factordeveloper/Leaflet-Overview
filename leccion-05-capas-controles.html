🗂️ Capas y Controles

Aprende a organizar tu mapa en capas (layers), usar grupos de capas (Layer Groups), y añadir controles interactivos para que el usuario pueda cambiar entre diferentes vistas y capas de información.

🗺️ Mapa con Control de Capas

🎛️ Controles disponibles:

  • Usa el control de capas (esquina superior derecha) para cambiar el mapa base.
  • Activa y desactivar diferentes capas de información (Museos, Parques y Vistas).
  • El control de escala está en la esquina inferior izquierda.

🎨 Capas Base (Base Layers)

¿Qué son las capas base?

Son los mapas de fondo. Solo puede estar activa UNA a la vez.

// Definir diferentes capas base const osm = L.tileLayer( 'https://tile.openstreetmap.org/{z}/{x}/{y}.png', { attribution: '© OpenStreetMap' } ); const dark = L.tileLayer( 'https://{s}.basemaps.cartocdn.com/dark_all/{z}/{x}/{y}.png', { attribution: '© CARTO' } ); // Objeto con las capas base const capasBase = { "OpenStreetMap": osm, "Modo Oscuro": dark }; // Añadir al control de capas L.control.layers(capasBase).addTo(mapa);

📚 Layer Groups (Grupos)

Agrupar elementos relacionados

Puedes agrupar múltiples marcadores o formas para controlarlos juntos:

// 2. Capas de Superposición (Overlays) const museos = L.layerGroup([ L.marker([4.6015, -74.0722]).bindPopup('Museo del Oro'), L.marker([4.5968, -74.0714]).bindPopup('Museo Botero') ]); const parques = L.layerGroup([ L.marker([4.6226, -74.0664]).bindPopup('Parque Nacional'), L.marker([4.6583, -74.0939]).bindPopup('Parque Simón Bolívar') ]);

🎛️ Control de Capas

Crear el control

El control permite al usuario cambiar capas:

// Overlays (pueden estar varias activas) const overlays = { "🏛️ Museos": museos, "🌳 Parques": parques, "🔭 Vistas": vistasLayer }; // Crear y añadir el control L.control.layers(capasBase, overlays, { collapsed: false, // Mantener expandido position: 'topright' }).addTo(mapa);
