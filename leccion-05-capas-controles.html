🗂️ Capas y Controles

Aprende a organizar tu mapa en capas (layers), usar grupos de capas (Layer Groups), y añadir controles interactivos para que el usuario pueda cambiar entre diferentes vistas y capas de información.

🗺️ Mapa con Control de Capas

🎛️ Controles disponibles:

  • Usa el control de capas (esquina superior derecha) para cambiar el mapa base
  • Activa/desactiva las capas de información (overlays)
  • El control de escala está en la esquina inferior izquierda

🎨 Capas Base (Base Layers)

¿Qué son las capas base?

Son los mapas de fondo. Solo puede estar activa UNA a la vez (como estaciones de radio).

// Definir diferentes capas base const osm = L.tileLayer( 'https://tile.openstreetmap.org/{z}/{x}/{y}.png', { attribution: '© OpenStreetMap' } ); const cartoDB = L.tileLayer( 'https://{s}.basemaps.cartocdn.com/light_all/{z}/{x}/{y}.png', { attribution: '© CartoDB' } ); const dark = L.tileLayer( 'https://{s}.basemaps.cartocdn.com/dark_all/{z}/{x}/{y}.png', { attribution: '© CartoDB' } ); // Objeto con las capas base const capasBase = { "OpenStreetMap": osm, "CartoDB Claro": cartoDB, "CartoDB Oscuro": dark };
💡 Tip: El nombre que usas en el objeto es el que aparecerá en el control de capas.

📚 Layer Groups (Grupos)

Agrupar elementos relacionados

Puedes agrupar múltiples marcadores o formas para controlarlos juntos:

// Crear marcadores const restaurante1 = L.marker([40.42, -3.71]) .bindPopup('Restaurante A'); const restaurante2 = L.marker([40.41, -3.70]) .bindPopup('Restaurante B'); // Agruparlos en un Layer Group const restaurantes = L.layerGroup([ restaurante1, restaurante2 ]); // Añadir todo el grupo al mapa restaurantes.addTo(mapa); // Puedes añadir más elementos después const restaurante3 = L.marker([40.415, -3.695]); restaurantes.addLayer(restaurante3); // O eliminar elementos restaurantes.removeLayer(restaurante1); // Limpiar todo el grupo restaurantes.clearLayers();

Feature Group

Similar a LayerGroup pero con métodos adicionales:

const zonas = L.featureGroup([ L.polygon([[40.42,-3.71], [40.41,-3.72], [40.40,-3.70]]), L.circle([40.43, -3.69], {radius: 300}) ]).addTo(mapa); // FeatureGroup permite: zonas.getBounds(); // Obtener límites del grupo zonas.bindPopup('Todas las zonas'); // Popup para todo // Ajustar vista a todo el grupo mapa.fitBounds(zonas.getBounds());

🎛️ Control de Capas

Crear el control

El control permite al usuario cambiar capas:

// Capas base (solo una activa) const capasBase = { "OpenStreetMap": osm, "CartoDB": cartoDB }; // Overlays (pueden estar varias activas) const overlays = { "🍽️ Restaurantes": restaurantes, "🏨 Hoteles": hoteles, "🚌 Transporte": transporte }; // Crear y añadir el control L.control.layers(capasBase, overlays, { collapsed: false, // Expandido por defecto position: 'topright' // Posición }).addTo(mapa);
⚠️ Importante: Las capas base usan radio buttons (solo una), los overlays usan checkboxes (múltiples).

Otros controles

// Control de escala L.control.scale({ metric: true, // Mostrar metros/km imperial: false, // Ocultar millas position: 'bottomleft' }).addTo(mapa); // Control de zoom (se añade por defecto) L.control.zoom({ position: 'topleft', zoomInTitle: 'Acercar', zoomOutTitle: 'Alejar' }).addTo(mapa); // Control de atribución L.control.attribution({ prefix: 'Mi App' }).addTo(mapa);

🔧 Control Personalizado

// Crear un control personalizado const MiControl = L.Control.extend({ options: { position: 'bottomright' }, onAdd: function(map) { // Crear el contenedor del control const container = L.DomUtil.create('div', 'mi-control'); container.innerHTML = '<button onclick="centrarMapa()">🎯 Centrar</button>'; // Evitar que clics propaguen al mapa L.DomEvent.disableClickPropagation(container); return container; } }); // Instanciar y añadir new MiControl().addTo(mapa); // Función del botón function centrarMapa() { mapa.setView([40.4168, -3.7038], 13); }
