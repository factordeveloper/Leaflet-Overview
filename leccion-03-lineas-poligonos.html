📐 Líneas y Polígonos

Aprende a dibujar formas geométricas en el mapa: líneas (polylines), polígonos, rectángulos y círculos. Estas formas son esenciales para mostrar rutas, delimitar áreas, zonas de cobertura y mucho más.

🗺️ Formas Geométricas

Línea (El Septimazo)
Polígono (La Candelaria)
Rectángulo (Parque Nacional)
Círculo (Zona Colpatria)

📏 Polyline (Líneas)

Crear una línea

Una polyline conecta múltiples puntos formando una línea:

// Array de coordenadas [lat, lng] const puntos = [ [4.5981, -74.0760], // Plaza de Bolívar [4.6100, -74.0710], // Carrera 7 con Calle 26 [4.6226, -74.0664] // Parque Nacional ]; // Crear la línea const linea = L.polyline(puntos, { color: 'red', // Color de la línea weight: 4, // Grosor en píxeles opacity: 0.8, // Transparencia (0-1) dashArray: '10, 10', // Línea discontinua lineCap: 'round', // Extremos redondeados lineJoin: 'round' // Uniones redondeadas }).addTo(mapa); // Ajustar vista a la línea mapa.fitBounds(linea.getBounds());
💡 Tip: Usa fitBounds() para ajustar automáticamente el zoom del mapa y mostrar toda la forma.

🔷 Polygon (Polígono)

Crear un polígono

Un polígono es una forma cerrada con relleno:

const coordenadas = [ [4.6000, -74.0780], [4.5950, -74.0780], [4.5950, -74.0720], [4.6000, -74.0720] ]; const poligono = L.polygon(coordenadas, { color: 'blue', // Color del borde weight: 3, // Grosor del borde fillColor: '#3498db',// Color de relleno fillOpacity: 0.4 // Transparencia relleno }).addTo(mapa); // Añadir popup al polígono poligono.bindPopup('Barrio La Candelaria');

Polígono con agujero

// Polígono exterior y agujero interior const conAgujero = [ [ // Contorno exterior [40.43, -3.72], [40.43, -3.68], [40.40, -3.68], [40.40, -3.72] ], [ // Agujero interior [40.42, -3.71], [40.42, -3.69], [40.41, -3.69], [40.41, -3.71] ] ]; L.polygon(conAgujero).addTo(mapa);

⬜ Rectángulo y ⭕ Círculo

Crear un rectángulo

Se define con dos esquinas opuestas:

// Esquinas del rectángulo const esquinas = [ [4.6250, -74.0700], // Esquina superior-izq [4.6180, -74.0620] // Esquina inferior-der ]; const rectangulo = L.rectangle(esquinas, { color: '#2ecc71', weight: 2, fillColor: '#2ecc71', fillOpacity: 0.3 }).addTo(mapa);

Crear un círculo

Se define con centro y radio en metros:

const circulo = L.circle( [4.6103, -74.0706], // Torre Colpatria { radius: 500, // Radio en METROS color: '#9b59b6', fillColor: '#9b59b6', fillOpacity: 0.3 } ).addTo(mapa); // CircleMarker: radio en PÍXELES (no cambia con zoom) L.circleMarker([40.41, -3.68], { radius: 10 // 10 píxeles }).addTo(mapa);
💡 Diferencia: L.circle() usa metros (escala con zoom), mientras que L.circleMarker() usa píxeles (tamaño fijo en pantalla).

🎨 Opciones de Estilo Comunes

Propiedades disponibles para todas las formas:

  • stroke: true/false - Mostrar borde
  • color: Color del borde (hex, nombre, rgb)
  • weight: Grosor del borde en píxeles
  • opacity: Transparencia del borde (0-1)
  • fill: true/false - Mostrar relleno
  • fillColor: Color del relleno
  • fillOpacity: Transparencia del relleno (0-1)
  • dashArray: Patrón de línea discontinua (ej: '5, 10')
  • className: Clase CSS personalizada
