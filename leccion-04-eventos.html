🎯 Eventos del Mapa

Los eventos permiten que tu mapa responda a las acciones del usuario: clics, movimientos del ratón, cambios de zoom, etc. Aprenderás a escuchar y manejar estos eventos para crear mapas verdaderamente interactivos.

Eventos que estamos escuchando:

  • click: Clic en el mapa (añade marcador)
  • dblclick: Doble clic
  • mousemove: Movimiento del ratón
  • zoomend: Fin de cambio de zoom
  • moveend: Fin de movimiento del mapa
  • contextmenu: Clic derecho

🖱️ Eventos del Mapa

Evento click

El más común: detectar dónde hizo clic el usuario:

mapa.on('click', function(e) { const lat = e.latlng.lat; const lng = e.latlng.lng; L.marker([lat, lng]) .addTo(mapa) .bindPopup(`Coordenadas en Bogotá:<br>Lat: ${lat.toFixed(4)}<br>Lng: ${lng.toFixed(4)}`) .openPopup(); });

Evento mousemove

Seguir la posición del cursor:

mapa.on('mousemove', function(e) { const coords = e.latlng; document.getElementById('display').innerHTML = `Lat: ${coords.lat.toFixed(4)}, Lng: ${coords.lng.toFixed(4)}`; });

Eventos de zoom y movimiento

// Cuando cambia el zoom mapa.on('zoomend', function() { console.log('Zoom actual:', mapa.getZoom()); }); // Cuando termina de moverse el mapa mapa.on('moveend', function() { const centro = mapa.getCenter(); console.log('Centro:', centro); });

📍 Eventos en Marcadores

Eventos comunes en marcadores

const marcador = L.marker([40.42, -3.70]).addTo(mapa); // Clic en el marcador marcador.on('click', function(e) { console.log('Marcador clickeado!'); }); // Pasar ratón por encima marcador.on('mouseover', function() { this.openPopup(); }); // Sacar ratón del marcador marcador.on('mouseout', function() { this.closePopup(); }); // Si es arrastrable marcador.on('dragend', function(e) { const pos = e.target.getLatLng(); console.log('Nueva posición:', pos); });
💡 Tip: Usa once() en lugar de on() si quieres que el evento se ejecute solo una vez.

Eliminar eventos

// Definir la función del evento function miFuncion(e) { console.log('Evento!'); } // Añadir evento mapa.on('click', miFuncion); // Eliminar evento específico mapa.off('click', miFuncion); // Eliminar TODOS los eventos de un tipo mapa.off('click');

📚 Lista de Eventos Disponibles

Eventos del Mapa

  • click, dblclick: Clics
  • mousedown, mouseup: Presionar/soltar ratón
  • mouseover, mouseout: Entrar/salir del mapa
  • mousemove: Movimiento del ratón
  • contextmenu: Clic derecho
  • keypress, keydown, keyup: Teclado
  • zoom, zoomstart, zoomend: Zoom
  • move, movestart, moveend: Movimiento
  • resize: Cambio de tamaño
  • load: Carga inicial

Eventos de Capas/Marcadores

  • add: Al añadir al mapa
  • remove: Al eliminar del mapa
  • popupopen, popupclose: Popups
  • tooltipopen, tooltipclose: Tooltips
  • drag, dragstart, dragend: Arrastrar
