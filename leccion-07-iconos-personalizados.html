🖼️ L.Icon - Icono con imagen

Crear icono desde imagen

Usa tus propias imágenes PNG/SVG como marcadores:

// Crear icono personalizado const miIcono = L.icon({ iconUrl: 'imagenes/mi-marcador.png', iconSize: [32, 32], // Tamaño del icono [ancho, alto] iconAnchor: [16, 32], // Punto de anclaje (centro-abajo) popupAnchor: [0, -32], // Dónde aparece el popup shadowUrl: 'imagenes/sombra.png', shadowSize: [41, 41], shadowAnchor: [12, 41] }); // Usar el icono en un marcador L.marker([40.4168, -3.7038], { icon: miIcono }).addTo(mapa);
💡 iconAnchor: Define qué punto del icono se coloca en las coordenadas. [16, 32] = centro horizontal, abajo vertical (para un pin).

Crear clase de icono reutilizable

// Extender Icon para crear una clase const IconoCategoria = L.Icon.extend({ options: { iconSize: [32, 32], iconAnchor: [16, 32], popupAnchor: [0, -32] } }); // Crear instancias con diferentes imágenes const iconoHotel = new IconoCategoria({iconUrl: 'hotel.png'}); const iconoRestaurante = new IconoCategoria({iconUrl: 'resto.png'});

📝 L.DivIcon - Icono HTML/CSS

Iconos con HTML puro

Crea iconos dinámicos sin necesidad de imágenes:

// Icono con emoji const iconoEmoji = L.divIcon({ html: '<div class="emoji-marker">📍</div>', className: 'custom-div-icon', iconSize: [30, 30], iconAnchor: [15, 30] }); L.marker([40.42, -3.71], {icon: iconoEmoji}).addTo(mapa);

Icono numerado dinámico

// Función para crear iconos numerados function iconoNumerado(numero, color) { return L.divIcon({ html: ` <div style=" width: 30px; height: 30px; background: ${color}; border-radius: 50%; border: 2px solid white; display: flex; align-items: center; justify-content: center; color: white; font-weight: bold; box-shadow: 0 2px 5px rgba(0,0,0,0.3); ">${numero}</div> `, className: '', iconSize: [30, 30], iconAnchor: [15, 15] }); } // Crear marcadores numerados L.marker([40.42, -3.71], {icon: iconoNumerado(1, '#e74c3c')}); L.marker([40.41, -3.70], {icon: iconoNumerado(2, '#3498db')});

✨ Iconos Animados

Efecto de pulso con CSS

/* CSS para animación de pulso */ .pulse-marker { width: 20px; height: 20px; background: #e74c3c; border-radius: 50%; position: relative; } .pulse-marker::before { content: ''; position: absolute; width: 100%; height: 100%; background: #e74c3c; border-radius: 50%; animation: pulse 2s infinite; } @keyframes pulse { 0% { transform: scale(1); opacity: 1; } 100% { transform: scale(3); opacity: 0; } }
// Crear icono pulsante const iconoPulso = L.divIcon({ html: '<div class="pulse-marker"></div>', className: '', iconSize: [20, 20], iconAnchor: [10, 10] }); L.marker([40.42, -3.70], {icon: iconoPulso}).addTo(mapa);

Icono con rebote

/* CSS para rebote */ @keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-10px); } } .bounce-marker { animation: bounce 1s infinite; }
💡 Tip: Usa animaciones CSS con moderación para no sobrecargar el mapa, especialmente con muchos marcadores.

🎯 Ejemplo Completo: Iconos por Categoría

// Definir iconos por categoría const iconos = { restaurante: L.divIcon({ html: '<div class="emoji-marker">🍽️</div>', className: '', iconSize: [30, 30], iconAnchor: [15, 15] }), hotel: L.divIcon({ html: '<div class="emoji-marker">🏨</div>', className: '', iconSize: [30, 30], iconAnchor: [15, 15] }), museo: L.divIcon({ html: '<div class="emoji-marker">🏛️</div>', className: '', iconSize: [30, 30], iconAnchor: [15, 15] }), transporte: L.divIcon({ html: '<div class="emoji-marker">🚇</div>', className: '', iconSize: [30, 30], iconAnchor: [15, 15] }) }; // Datos de lugares const lugares = [ { lat: 40.42, lng: -3.71, nombre: 'Casa Botín', categoria: 'restaurante' }, { lat: 40.41, lng: -3.69, nombre: 'Hotel Ritz', categoria: 'hotel' }, { lat: 40.414, lng: -3.692, nombre: 'Museo del Prado', categoria: 'museo' } ]; // Crear marcadores con iconos según categoría lugares.forEach(lugar => { L.marker([lugar.lat, lugar.lng], { icon: iconos[lugar.categoria] }) .bindPopup(`<b>${lugar.nombre}</b><br>${lugar.categoria}`) .addTo(mapa); });
