📊 GeoJSON

GeoJSON es un formato estándar para representar datos geográficos. Es la forma más común de intercambiar datos entre aplicaciones de mapas. Aprenderás a cargar, estilizar y manipular datos GeoJSON en Leaflet.

📄 Estructura GeoJSON

¿Qué es GeoJSON?

Es JSON con estructura específica para datos geográficos. Tiene 3 tipos principales:

  • Point: Un punto (marcador)
  • LineString: Una línea
  • Polygon: Un polígono
// Estructura de un Feature (elemento) { "type": "Feature", "properties": { "nombre": "Mi lugar", "categoria": "parque" }, "geometry": { "type": "Point", "coordinates": [-3.7038, 40.4168] } }
⚠️ Importante: En GeoJSON las coordenadas son [longitud, latitud], ¡al revés que en Leaflet! Leaflet lo convierte automáticamente.
// FeatureCollection (colección de elementos) { "type": "FeatureCollection", "features": [ { /* Feature 1 */ }, { /* Feature 2 */ }, { /* ... */ } ] }

🔧 Cargar GeoJSON

Cargar datos directamente

// Datos GeoJSON como variable const datos = { "type": "FeatureCollection", "features": [ { "type": "Feature", "properties": { "name": "Retiro" }, "geometry": { "type": "Polygon", "coordinates": [[...]] } } ] }; // Añadir al mapa L.geoJSON(datos).addTo(mapa);

Cargar desde archivo/URL

// Usando fetch fetch('datos.geojson') .then(response => response.json()) .then(data => { L.geoJSON(data).addTo(mapa); }); // Con async/await async function cargarDatos() { const response = await fetch('datos.geojson'); const data = await response.json(); L.geoJSON(data).addTo(mapa); }

🎨 Estilizar GeoJSON

Estilos estáticos

L.geoJSON(datos, { style: { color: '#e74c3c', weight: 2, fillColor: '#e74c3c', fillOpacity: 0.3 } }).addTo(mapa);

Estilos dinámicos (según propiedades)

L.geoJSON(datos, { style: function(feature) { // Color según categoría const colores = { 'parque': '#2ecc71', 'edificio': '#3498db', 'zona': '#e74c3c' }; return { color: colores[feature.properties.categoria], weight: 2, fillOpacity: 0.4 }; } }).addTo(mapa);

Personalizar marcadores (puntos)

L.geoJSON(puntos, { pointToLayer: function(feature, latlng) { // Usar circleMarker en vez de marcador return L.circleMarker(latlng, { radius: 8, fillColor: '#3498db', color: 'white', weight: 2, fillOpacity: 0.8 }); } }).addTo(mapa);

⚡ Interactividad con GeoJSON

onEachFeature - Eventos por elemento

L.geoJSON(datos, { onEachFeature: function(feature, layer) { // Añadir popup con propiedades if (feature.properties) { layer.bindPopup( `<b>${feature.properties.nombre}</b>` ); } // Añadir eventos layer.on({ mouseover: function(e) { e.target.setStyle({ weight: 4, fillOpacity: 0.7 }); }, mouseout: function(e) { geojsonLayer.resetStyle(e.target); }, click: function(e) { mapa.fitBounds(e.target.getBounds()); } }); } }).addTo(mapa);

filter - Filtrar elementos

// Solo mostrar parques grandes L.geoJSON(datos, { filter: function(feature) { return feature.properties.tipo === 'parque' && feature.properties.area > 10000; } }).addTo(mapa);

Métodos útiles de la capa GeoJSON

const geojsonLayer = L.geoJSON(datos).addTo(mapa); // Añadir más datos geojsonLayer.addData(nuevosDatos); // Obtener límites mapa.fitBounds(geojsonLayer.getBounds()); // Iterar sobre elementos geojsonLayer.eachLayer(function(layer) { console.log(layer.feature.properties); }); // Limpiar todos los datos geojsonLayer.clearLayers();
